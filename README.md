# OpenClaw Skills Collection

老张头的 OpenClaw 技能集合仓库

## 📦 包含的技能

### 1. image-generate - 图片生成
支持多种图片生成模型（火山引擎豆包、Gemini 3 Pro）

**功能**：
- ✅ 文本生成图片
- ✅ 多模型支持（豆包 Seedream、Gemini 3 Pro）
- ✅ 环境变量配置，安全可靠
- ✅ 自动保存生成的图片

[查看详情](./image-generate/README.md)

### 2. openclaw_stock_research - 股票分析
专业的 A 股和港股分析系统，基于 akshare 开源数据库

**功能**：
- ✅ 技术面分析（K线、均线、MACD、RSI等）
- ✅ 基本面分析（财务数据、估值指标）
- ✅ 资金流向分析（主力资金、北向资金）
- ✅ 新闻面分析（实时新闻采集）
- ✅ 短期/长期选股策略
- ✅ 综合评分模型
- ✅ 实时预警系统

[查看详情](./openclaw_stock_research/SKILL.md)

### 3. video-generate - 视频生成
文本生成视频功能，支持首帧图片

**功能**：
- ✅ 文本描述生成视频
- ✅ 支持自定义首帧图片
- ✅ 火山引擎视频生成模型

[查看详情](./video-generate/SKILL.md)

## 🚀 使用方法

每个技能都有独立的 `SKILL.md` 或 `README.md` 文档，请查看对应目录。
